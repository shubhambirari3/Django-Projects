# Django Items Project

## Description
This project is a Django-based web application for managing items. Users can create, update, delete, and list items. User authentication is required for most actions.

## Features
- User authentication (login, logout, signup)
- Create, update, delete, and list items

## Commands
- To run the server: `python manage.py runserver`
- To apply migrations: `python manage.py migrate`

## URL Links
- Signup Page: http://localhost:8000/signup/
